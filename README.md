
# Liver Cancer Risk Assessment using Machine Learning

An AI-powered web application that predicts the **risk of liver cancer** based on patient demographics, lifestyle factors, and medical history.  
This project is developed as a **capstone / academic project** using **Machine Learning and Streamlit**.

---

## 📌 Project Overview

Liver cancer is a major global health concern, often detected at late stages.  
This project aims to assist in **early risk assessment** by leveraging machine learning models trained on clinical and lifestyle data.

The application:
- Takes **patient information** and **medical details**
- Predicts the **probability of liver cancer risk**
- Provides **preventive tips and safety measures** based on prediction

---

## 🚀 Features

- 🧠 Machine Learning–based risk prediction  
- 🖥️ Interactive and modern **Streamlit web interface**  
- 📊 Probability-based classification (High Risk / Low Risk)  
- 🛡️ Safety measures & lifestyle recommendations  
- 🎨 Dark themed, medical-grade UI  

---

## 🧪 Input Parameters

### 👤 Patient Information
- Age  
- Gender  
- Body Mass Index (BMI)  
- Physical Activity Level  
- Smoking Status  
- Alcohol Consumption  

### 🧬 Medical Information
- Alpha Fetoprotein (AFP)  
- Hepatitis B  
- Hepatitis C  
- Cirrhosis History  
- Diabetes  
- Family History of Cancer  

---

## 📈 Output

- **Risk Probability Score**
- **Risk Category**
  - ✅ Low Risk
  - ⚠️ High Risk
- **Preventive Tips & Safety Measures**
